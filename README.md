# Finanzas Cuantitativas con Python
Este repositorio contiene una colección de documentos y recursos relacionados con las finanzas cuantitativas utilizando Python. Aquí encontrarás guías detalladas sobre diversas metodologías y técnicas empleadas en la gestión de riesgos financieros, análisis de carteras y modelado estadístico.

## Contenido

### Riesgo de Mercado con Python: 

Una guía completa sobre el cálculo de métricas de riesgo de mercado, incluyendo volatilidad, Value at Risk (VaR) y Tracking Error, con ejemplos prácticos en Python. [Ver documento](Riesgo%20de%20Mercado/README.md)
